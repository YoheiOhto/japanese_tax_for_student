# japanese_tax_for_student

## 概要

このプロジェクトは、日本の税金制度、特に大学生や大学院生が直面する可能性のある税金について、Pythonで理解を深めることを目的としています。
給与所得・雑所得・奨学金などの収入から、所得税・住民税・国民健康保険・国民年金・各種控除を計算し、手取り額や確定申告での還付/追納額をシミュレーションできます。

## 主な機能

* **手取り計算シミュレーション**: `src/simulator.py` の `simulate_tax()` で、給与・雑所得・奨学金を合算した総収入から、所得税・住民税・国民年金・国民健康保険・社会保険料控除・各種所得控除・税額控除を差し引いた手取り額を計算します。
* **対応している控除・計算項目**:
    * 給与所得控除・基礎控除（所得税・住民税）・調整控除（住民税）
    * 社会保険料控除（国民健康保険・国民年金）
    * iDeCo（小規模企業共済等掛金控除）— 被保険者区分ごとの上限に対応
    * 青色申告特別控除（雑所得）
    * 勤労学生控除
    * 医療費控除 / セルフメディケーション税制（有利な方を自動選択）
    * 生命保険料控除・地震保険料控除（新制度）
    * 扶養控除の判定（扶養から外れるラインの可視化）・配偶者控除/配偶者特別控除
    * ふるさと納税（寄附金控除、確定申告ベース）
    * 源泉徴収額の概算と、確定申告による還付/追納額の試算

## 想定ユーザーとシナリオ

このプロジェクトは、特に以下の状況にある大学生・大学院生を主な対象としています。

* 国民健康保険・国民年金に加入している（会社の社会保険に未加入）
* 39歳以下
* バイト先・収入源が複数ある場合も想定（給与複数勤務先・雑所得・奨学金）

## 動作確認済みの年度

| 制度 | 対応年度 | 備考 |
| --- | --- | --- |
| 所得税の税率ブラケット | 全年度共通 | 令和7年度改正でも変更なし |
| 給与所得控除・基礎控除（所得税） | 2024年以前 / 2025〜2026年 / 2027年以降 | 令和7年度改正の経過措置に対応 |
| 基礎控除（住民税）・調整控除 | 全年度共通 | 現行制度では年度間で値の変動なし |
| 扶養控除の所得要件 | 2024年以前（48万円） / 2025年以降（58万円） | |
| 国民健康保険料率（文京区） | 2025年度・2026年度 | それ以外の年度を指定した場合は最新の既知年度（2026年度）の料率で概算し、結果に `"概算フラグ": true` を付与します |
| 国民年金保険料 | 2025年度・2026年度 | 同上、最新の既知年度（2026年度）の額で概算します |

国民健康保険・国民年金の料率は法改正・改定のたびに `src/config.py` の `KOKUHO_RATES` / `src/kokumin_nenkin.py` の `_NENKIN_RATES` に新年度のデータを追加することで更新できます。

## 出力項目の説明（用語集）

`simulate_tax()` の戻り値・Web アプリの「詳細内訳」に表示される主な項目の意味です。税に詳しくない方向けの簡単な説明です。

| 項目 | 説明 |
| --- | --- |
| 給与所得 | 給与収入から「給与所得控除」（経費の概算控除）を差し引いた金額 |
| 雑所得 | 副業・フリーランス収入や貸与型奨学金などの収入から必要経費を差し引いた金額 |
| 合計所得金額 | 給与所得＋雑所得など、各種所得控除を差し引く前の所得の合計 |
| 基礎控除 | 全員が無条件で受けられる所得控除。所得税と住民税で金額が異なる |
| 調整控除（住民税） | 所得税と住民税の基礎控除額の差を調整するための住民税の控除 |
| 社会保険料控除 | 支払った国民健康保険料・国民年金保険料の全額が所得控除になる |
| iDeCo控除 | iDeCo（個人型確定拠出年金）の掛金が全額所得控除になる（被保険者区分ごとに上限あり） |
| 勤労学生控除 | 学生で合計所得が一定以下の場合に受けられる追加の所得控除 |
| 医療費控除 / セルフメディケーション税制 | 年間の医療費・OTC医薬品購入費が一定額を超えた場合の所得控除（いずれか有利な方を採用） |
| 生命保険料控除・地震保険料控除 | 支払った保険料に応じた所得控除 |
| 配偶者控除 | 配偶者の所得が一定以下の場合に受けられる所得控除 |
| 扶養判定 | 自分の所得が、親などの扶養親族の所得要件（2025年以降は58万円以下）を満たすかどうかの判定。満たさなくなると親の税負担が増える |
| ふるさと納税控除 | ふるさと納税（寄附金控除）による所得税・住民税からの控除額。自己負担額は原則2,000円 |
| 源泉徴収額 | 給与から天引きされている所得税の概算額 |
| 還付額 / 追納額 | 確定申告をした場合に、源泉徴収額と実際の所得税額との差額として戻ってくる（または追加で納める）金額の概算 |

## ファイル構成

* `README.md`: このファイル
* `app.py`: Streamlit Web アプリ
* `cli.py`: コマンドラインインターフェース（argparse）
* `ohto_simulate.py`: `src.simulator` を使った実行例スクリプト
* `src/`: 各税金・控除の計算ロジックを実装したPythonモジュール群
    * `config.py`: 税率・控除額・国民健康保険料率などの定数
    * `simulator.py`: `SimulationInput` / `simulate_tax()` による統合シミュレーション
* `tests/`: 単体テスト（`uv run pytest` で実行）
* `archives/`: 旧実装（ノートブック・スクリプト）。詳細は `archives/README.md` を参照

## 使い方

### A. Web アプリ（推奨）

ブラウザ上で収入や控除の条件を入力するだけでリアルタイムに手取りを確認できます。

```bash
git clone https://github.com/YoheiOhto/japanese_tax_for_student.git
cd japanese_tax_for_student
uv sync
uv run streamlit run app.py
```

ブラウザで `http://localhost:8501` が自動的に開きます。

**機能:**
- サイドバーで給与収入（複数勤務先）・雑所得・奨学金・自治体・世帯人数・年齢・iDeCo・勤労学生控除・医療費控除・生命保険料控除/地震保険料控除・ふるさと納税・配偶者の所得などを入力
- 収入→手取りのウォーターフォールグラフ・支出割合の円グラフを表示
- 詳細内訳テーブル（各種控除額、還付/追納見込額、扶養判定）を表示
- 収入額を動かしたときの手取り変化（103万・130万・160万・201万の壁、扶養から外れる収入ラインを表示）
- 2024〜2027年の年度別比較

### B. CLI（コマンドライン）

```bash
uv run python cli.py --salary 200 --salary 50 --year 2026
uv run python cli.py --salary 96 --student --age 20
uv run python cli.py --salary 400 --furusato-kifu 4 --save --label case1
```

主なオプションは `uv run python cli.py --help` で確認できます。`--summary-only` でサマリーのみを表示、`--save` で `output/` に結果（JSON/CSV）を保存します。

結果は `output/` ディレクトリに JSON・CSV で保存されます。

## テスト

```bash
uv run pytest
```

所得税の税率ブラケット境界値（195万・330万・695万・900万・1,800万・4,000万円）など、国税庁の計算例と照合した単体テストを `tests/` に収録しています。
